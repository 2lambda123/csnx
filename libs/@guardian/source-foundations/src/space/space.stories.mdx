# Space

Space units create visual rhythm and balance

## Example

```tsx
import { space } from '@guardian/source-foundations';

const container = css`
	margin: ${space[1]}px;
`;

/*
.class-name {
  margin: 4px;
}
*/
```

## Space scale

The following units can be applied to margin or padding properties, vertically
or horizontally.

- `space[1]` -> 4px
- `space[2]` -> 8px
- `space[3]` -> 12px
- `space[4]` -> 16px
- `space[5]` -> 20px
- `space[6]` -> 24px
- `space[9]` -> 36px
- `space[12]` -> 48px
- `space[24]` -> 96px
